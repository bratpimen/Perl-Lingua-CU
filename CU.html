NAME

Lingua::CU - Perl extension for working with Church Slavonic text in Unicode

SYNOPSIS

  use Lingua::CU ':all';
  arabicToCyrillic (21); # returns к҃а
  cyrillicToArabic ("к҃а"); # returns 21
  resolve ("ст҃ъ"); # returns свѧ́тъ
  cu2ru ("ст҃ъ"); # returns свя́тъ
  cu2ru ("ст҃ъ", { noaccents => 1, modernrules => 1 }); # returns свят

DESCRIPTION

Lingua::CU is a module for performing various operations with Church Slavonic texts.

It includes the following capabilities:

  • Resolve Church Slavonic abbreviations and nomina sacra
  • Convert between Cyrillic and Arabic numerals
  • Perform basic conversions between the Julian and Gregorian calendars (TODO)
  • Convert Church Slavonic text to Russian characters (both traditional and reformed orthography)
  • Romanize (transliterate to Latin) Church Slavonic text using various systems (TODO)
  • Convert between Unicode and legacy UCS and HIP encodings (TODO)
  • Sort Church Slavonic words using a tailoring of the DUCET (TODO)
  • Perform stemming of Church Slavonic words (TODO)

All text supplied to this library must be encoded in UTF-8 and, unless otherwise specified, is assumed to be in Unicode. For more on Church Slavonic using Unicode, please see the paper Roadmap for Church Slavonic Typography in the Unicode Standard available at www.ponomar.net

EXPORT

No methods are exported by default.

The following methods may be exported if specified explicitly: arabicToCyrillic cyrillicToArabic resolve cu2ru.

You may also export all of the above methods by writing: use Lingua::CU ':all';

METHODS

arabicToCyrillic

Usage: arabicToCyrillic( $number )

Takes an Arabic number and returns the corresponding Cyrillic numeral. Croaks if $number is not numeric.

Example: arabicToCyrillic ( 121 ) returns рк҃а.

cyrillicToArabic

Usage: cyrillicToArabic( $numeral )

Takes a Cyrillic numeral and returns the corresponding Arabic number. Carps if $numeral is not a well-formatted Slavonic number.

Example: cyrillicToArabic("рк҃а") returns 121.

resolve

Usage: resolve( $word )

Takes a word that is written with a titlo or lettered titlo (as an abbreviation or nomen sacrum) and writes it out in full, resolving the abbreviation.

Bugs: correct placement of stress marks and capitalization are not guaranteed. Titlo resolution relies on a list that can still be improved.

Warning: the Slavonic word сн҃а could both be an abbreviation for Сы́на and a numeral (251). Thus, resolve('сн҃а') will return Сы́на but cyrillicToArabic('сн҃а') will return 251.

cu2ru

Usage: cu2ru( $text, [modernrules = 0, noaccents = 0])

Takes well-formatted Church Slavonic $text and transforms it into Russian (civil) orthography. The following operations are performed:

  • Titli and lettered titli are resolved
  • Cyrillic numerals are resolved to Arabic numbers (but see note above concerning сн҃а)
  • Stress marks are transformed to the acute accent (U+0301) and all other diacritical marks are removed
  • Letters that do not occur in Russian are transformed into their Russian analogs (e.g., ѧ is transformed to я)
  • Some spelling is normalized to agree with common Russian rules (e.g., шы is transformed to ши)
  • If optional parameter noaccents is not zero, all stress marks are removed; otherwise, stress indications remain in the text, but only as the acute accent (U+0301)
  • If optional parameter modernrules is not zero, the text is further simplified into modern Russian orthography (that means that і is resolved to и, ѣ is resolved to е, and trailing ъ is removed); otherwise, traditional (pre-1918) orthography is assumed
